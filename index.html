Antibody A faster and simpler antigen written in Golang.

You can download, extract it somewhere and source the antibody.zsh file from here and then script it somewhere: 

    
$ mkdir /somewhere/antibody
$ tar xzvf antibody.tar.gz -C /somewhere/antibody
$ source /somewhere/antibody/antibody.zsh
$ antibody

On OSX (or Linux if you use linuxbrew), you can install it using a homebrew tap: 

    
$ brew tap getantibody/homebrew-antibody
$ brew install antibody
$ source "$(brew --prefix)/share/antibody.zsh"
$ antibody

You can also always find more info in the readme.
