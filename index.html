Antibody A faster and simpler antigen written in Golang.

How much faster, exactly?

This chart demos the times taken to load 20 libraries 50 times:

Note that the first execution is slower because it is cloning all the repos.

Antibody is faster because it uses smarter git commands for that and can do stuff in parallel.

This also makes Antibody capable of saving of some disk space:

Instalation: 

    
$ curl -s https://raw.githubusercontent.com/getantibody/installer/master/install | bash -s
$ echo 'source <(antibody init)' >> ~/.zshrc
$ antibody

On OSX (or Linux if you use linuxbrew), you can also install it using a homebrew tap: 

    
$ brew tap getantibody/homebrew-antibody
$ brew install antibody
$ echo 'source <(antibody init)' >> ~/.zshrc
$ antibody
